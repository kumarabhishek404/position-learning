Following is POSITION Property of CSS

The position property specifies the type of positioning method used for an element.

Sticky

It is positioned based on the user's scroll position
This element stick at top ( top: 0; )

Fixed

It is positioned relative to viewport
This box is fixed at 300px from top and 40 px from left.

Ancestor of Absolute Box

Absolute

It is positioned relative to the nearest positioned ancestor
This box is at absolute position relative to its ancestor box,
10px from bottom and 0px from right of ancestor box.

Relative

It is positioned relative to its normal position.
This box is positioned relative to its normal position by 10px from top and 300px from left.

Static

Static positioned elements are not affected by the top, bottom, left, right. It is always positioned according to the normal flow of the page.
